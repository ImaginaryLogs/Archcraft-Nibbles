
# Instruction Set

|Hex|Opcode   |Notes                          |
|---|---------|-------------------------------|
|0x0|HALT     |Halts the Computer             |
|0x1|LOADA    |Loads A from IMM               |
|0x2|LOADB    |Loads B from IMM               |
|0x3|STOREA   |Stores A from IMM              |
|0x4|ADD      |A := A + B                     |
|0x5|SUB      |A := A - B                     |
|0x6|LOADOUT  |Loads A from IMM               |
|0x7|JUMPZERO |Jumps to IMM if zero-flag is on|
